# EI-PLACEMENT

Exercise-1 : Design Patterns--SmartHomeDemo

This project is a Java-based Smart Home System that shows how design patterns can make code modular and flexible. It uses the Builder pattern to configure the home setup, the Factory Method to create devices like Light, Fan, and Thermostat, the Adapter to connect a SmartTV, and the Decorator to add logging without changing device classes. For behavior, it applies the Observer pattern where a motion sensor triggers alerts and lights, and the Strategy pattern to apply different energy-saving modes. Together, these demonstrate how Creational, Structural, and Behavioral patterns work in a real-world scenario.

Exercise-2 : Astronaut Command System

This project is a Java simulation of a Satellite Control System for astronauts, demonstrating the use of design patterns in a command-based setup. It applies the Command pattern to execute and undo satellite actions like rotate, activate solar panels, and collect data, the Observer pattern to notify astronauts of live satellite updates, and the Singleton pattern to ensure only one control center instance exists. The program highlights modular design, event-driven updates, and clean OOP principles through a simple console-based interface.
